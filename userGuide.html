User Guide

The Safe User Guide explains how to install, configure, and use the Safe application. It is designed for families, caregivers, travelers, and anyone who needs fast, reliable emergency communication.

1. Creating & Managing User Profiles

  • Full name
  • Date of birth
  • Height & weight
  • Allergies
  • Medical history
  • Current medications
  • Blood type
  • Emergency contacts

How to Create a Profile

  1. Open the Safe app
  2. Tap the Profiles icon
  3. Select Add New Profile
  4. Enter required information
  5. Save changes

2. Navigation & Real-Time Location Features

  • Live GPS tracking
  • Last known location
  • Geofencing alerts
  • Movement history

How to Use

Open the menu and select Family Map. Choose a family member to view their current or last known location. Enable geofencing to receive alerts when someone enters or leaves a zone.

3. Emergency Alerts & SOS

What the SOS Button Sends

  • Real-time location
  • Medical profile
  • Distress alert message
  • Notifications to family and emergency services

How to Trigger SOS

  1. Press and hold the SOS button for 3 seconds
  2. A confirmation countdown appears
  3. The alert is sent automatically

4. Child Safety Tools

  • School-zone alerts
  • Trusted adult pickup authorization
  • Location history
  • Restricted geofencing zones

5. Travel Safety Mode

  • Local emergency numbers
  • Embassy contact information
  • High-risk area alerts
  • Travel document storage

6. Disaster Mode

  • Offline maps
  • Local shelter locations
  • Family rally point coordination
