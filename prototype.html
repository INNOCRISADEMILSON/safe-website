Safe App Prototype

The Safe prototype demonstrates the core layout, navigation flow, and essential safety features of the application. It provides a visual and functional preview of how families will interact with the platform during everyday use and emergency situations.

Main Screens in the Prototype

Home Dashboard

The dashboard provides quick access to the most important safety tools, including the SOS button, Family Map, Profiles, Travel Mode, Disaster Mode, and notifications.

Family Map

A real-time map showing active locations, last known locations, geofencing zones, and movement history. Users can tap a profile to view details or send a check-in request.

User Profiles

Each profile displays medical information, allergies, medications, emergency contacts, and a quick-share medical card. Profiles can be edited or updated at any time.

SOS Emergency Screen

The SOS interface includes long-press activation, a 3-second confirmation countdown, and automatic alert sending with real-time location and medical data.

Child Safety Tools

Includes school-zone alerts, trusted adult pickup authorization, restricted zones, and location history timelines.

Travel Safety Mode

Automatically adjusts to provide local emergency numbers, embassy contacts, high-risk area alerts, and travel document storage.

Disaster Mode

Provides offline maps, shelter locations, and family rally point coordination for use during network outages or natural disasters.

Prototype Goals

The Safe prototype is designed to demonstrate the app’s core features, show navigation flow, support testing, and ensure accessibility for all users.
